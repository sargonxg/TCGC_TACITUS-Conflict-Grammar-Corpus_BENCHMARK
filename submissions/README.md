# Submissions

Each submission lives under `submissions/<system-name>/<YYYY-MM-DD>/`.

Required files:
- `predictions.jsonl` — one JSON object per line, `id` field matching item ids
- `SYSTEM_CARD.md` — generated via `tcgc card scores.json`
- `submission.json` — metadata matching `schema/submission-v0.1.json`

See [Submission Guide](../docs/submission-guide.md) for the full process.
