# Roadmap

## Q4 2026 — Dataset paper

Full TCGC corpus paper submitted to a top-tier NLP venue. The paper covers: annotation methodology, ontology design decisions, IAA analysis, baseline system results, and open questions.

## Q1 2027 — OAG methodology

Publication of the Ontology-Anchored Generation (OAG) methodology — the system used to bootstrap TCGC item generation from the Conflict Grammar. This will include tooling for researchers to extend the corpus in new domains.

## Q2 2027 — TCGC v2

- Extended corpus: ~1,000+ items
- New task types: escalation-prediction, actor-alignment-shift
- Character-span provenance (IoU scoring)
- Multi-turn negotiation items
- Non-English items (pilot)

## Ongoing

- Leaderboard updates as submissions are received
- Ontology extensions via community proposal process (Tier 04)
- Full-corpus access via DUA (email hello@tacitus.me)
